# caesar-cipher-C#-Desktop
# 🔐 Encryption & Decryption System (C# Desktop App)

A simple yet powerful desktop application built using **C# WinForms** that demonstrates core encryption concepts including Caesar Cipher-like transformations and text encoding techniques.

This project was built as a learning exercise to strengthen my understanding of:
- Cryptography fundamentals
- Desktop application development
- Clean code structure in C#

---

## ✨ Features

- Encrypt plain text using custom logic
- Decrypt encrypted text back to original
- Convert text into binary representation
- Convert binary back to readable text
- Simple and user-friendly WinForms UI
- Modular code structure for easy maintenance

---

## 🧠 What I Learned

- How basic encryption algorithms work (Caesar-style logic)
- Structuring a multi-form C# application
- Handling user input securely and efficiently
- Separating logic from UI (basic architecture thinking)
- Managing project dependencies in Visual Studio

---

## 🛠️ Tech Stack

- Language: C#
- Framework: .NET Framework (WinForms)
